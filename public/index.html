Hackathon 2019
编码城市新生活
江苏无锡，2019年5月25日—— 26日

简介

本次黑客松活动由无锡市人才办和无锡市团委主办，依托无锡高新技术产业优势，立足青年创新创业，编码城市新生活，Let's code in Wuxi!

愿景

互联网、物联网、人工智能、大数据、区块链、云计算、5G……以计算机为核心的技术正深刻影响和改变着人类社会。电商、移动社交、共享单车……日常生活从未像今天一般如此贴近科技。
CodeWuxi号召每一位极客回归生活，发现身边点滴，用技术解决或改变日常生活中息息相关的问题。共同打造“编码城市新生活”！

主题

本次黑客松以“编码城市新生活”为目标，旨在鼓励参赛团队通过技术实现让城市生活变得更加美好。
参赛团队可以回归城市生活的方方面面，寻找可以通过技术优化改变的社会问题或生活方式。主题包括但不限于：
