关于CodeWuxi

CodeWuxi 最初发起于英国南安普顿大学，作为一只中国留学生组建的团队，基于无锡市政府开放数据，完成了一个基于LBS的城市环境评估系统。现在，在无锡市各政府部门的大力支持下，CodeWuxi 成为了一个坐标无锡，着眼互联网创新创业的活动品牌。CodeWuxi 也将不忘初衷，关注科技创新与城市发展的结合，秉承“科技让城市生活更美好”的理念，致力于开展面向高中生、大学生和专业领域的各类 hackathon（黑客松）、workshop（研习会）和 panel（专题讨论）等活动，普及科技成果，激发创新热情，重塑工匠精神。
